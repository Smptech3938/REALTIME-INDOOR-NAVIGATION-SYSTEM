# Indoor Wayfinder

Indoor Wayfinder is a Flutter-based indoor navigation application that combines TensorFlow Lite (TFLite) object recognition, OCR-based text detection, and A* pathfinding to provide real-time navigation assistance inside buildings.

## Features

* Real-time camera-based landmark recognition
* TensorFlow Lite model integration
* OCR fallback using Google ML Kit
* A* pathfinding algorithm for route generation
* Dynamic destination selection
* Indoor navigation guidance
* Flutter cross-platform architecture

## Tech Stack

* Flutter
* Dart
* TensorFlow Lite
* Google ML Kit OCR
* Camera Plugin
* A* Pathfinding

## Installation

1. Clone the repository
2. Run:

```bash
flutter pub get
```

3. Connect an Android device

4. Run:

```bash
flutter run
```

## Current Status

This project is currently in MVP development. The system supports object recognition, OCR-assisted localization, and indoor path navigation.
